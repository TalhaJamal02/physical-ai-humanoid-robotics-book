# Hardware Setup

This guide covers the recommended hardware for working through the examples in this book.

## Recommended Hardware

- **Workstation**: A desktop or laptop with a dedicated NVIDIA GPU is recommended for running the simulations and training the AI models.
  - **OS**: Ubuntu 22.04 LTS
  - **GPU**: NVIDIA RTX 3060 or higher
  - **RAM**: 32GB or more
  - **CPU**: Intel Core i7 or AMD Ryzen 7 or higher

- **Edge Device (Optional)**: For deploying the final project on a physical robot, a Jetson device is recommended.
  - **Device**: NVIDIA Jetson Orin Nano Developer Kit
