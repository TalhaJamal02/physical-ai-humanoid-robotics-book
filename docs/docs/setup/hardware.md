# Hardware Setup

This guide outlines the recommended hardware for running the examples in this book.

## Recommended Workstation

A desktop or laptop with a dedicated NVIDIA GPU is recommended for simulations and AI model training.

- **Operating System:** Ubuntu 22.04 LTS  
- **GPU:** NVIDIA RTX 3060 or higher  
- **RAM:** 32GB or more  
- **CPU:** Intel Core i7 or AMD Ryzen 7 or higher  

## Edge Device (Optional)

For deploying the final project on a physical robot, an NVIDIA Jetson device is recommended:

- **Device:** NVIDIA Jetson Orin Nano Developer Kit  
- Ideal for running ROS 2 nodes and lightweight AI inference on the robot.
